# Start Here Checklist (Path-locked)

1. Run `./scripts/doctor.sh`
   - Expected: ✅ for Node or Python
   - Common mistake: running from a different directory
   - Recovery: `cd <repo-root>` and re-run

2. Run `./scripts/verify.sh`
   - Expected: "Verify complete."
   - Common mistake: missing Node/npm
   - Recovery: install Node, rerun

3. Read `CLAUDE.md`
   - Expected: you understand non-negotiables and stop conditions
