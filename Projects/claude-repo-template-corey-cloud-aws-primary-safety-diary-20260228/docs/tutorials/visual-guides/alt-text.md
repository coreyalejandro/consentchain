# Visual Guide Text Alternatives (Spatial-substitution)

Any visual diagram must have a text equivalent.

Required fields:

1. Orientation cue ("Top is root", "Left is input", etc.)
2. Start point ("Begin at Root CLAUDE.md")
3. Ordered traversal ("Then go to .claude/rules, then hooks...")
4. If arrows exist, describe meaning ("enforces", "references", "blocks")

This is mandatory for accessibility and poor spatial reasoning support.
